Googly Eyes

Steps:

  1. Open GooglyEyes.pde using Processing.
  2. Choose a face image from the "img" folder in _2_googly_eyes, then drag and drop it into your processing window
  3. On line 1 of your sketch (outside of the methods) declare a variable for the image and the x and y values for the left eyeball: 
    PImage face;
int x;
int y;
  4. In your setup() method, import your image using the following code: 
    face = loadImage(“face.jpg”);
  5. Change the code you just entered so that “face.jpg” matches the name of the picture you chose.
  6. Set the size of your sketch and the size of your image to be the same by entering the following code in the setup method. 
    size(800,600);
face.resize(width,height);
  7. In the draw() method, place a white ellipse over the left eye of your image (like you did for Terminator Cat).
    HINT: To find out where to put it, add code to print the mouseX and mouseY where you click the mouse and use the x and y variables to save the numbers. 
    if(mousePressed){
    println("Mouse’s x-position: " + mouseX + "\n" +
     "Mouse’s y-position: " + mouseY + "\n");
}
  8. Now add a smaller black ellipse on the left eye to represent the pupil. Use the x and y values you saved earlier for the center
  9. Use mouseX and mouseY to make left pupil follow the mouse by putting this code inside the draw method:
    x = mouseX;
y = mouseY;
  10. Here’s the tricky part: we now need to stop the pupil from traveling outside of the eye. We are going to imagine a rectangle inside the white part of the eye (like the hit box we used in Where's Waldo).
    First, click on the top left corner and save the values in variables:
    int minX;
    int minY;

    Next, click on the bottom right corner and save the values in variables:
    int maxX;
    int maxY;

  11. Now we need code to keep the pupil inside the hit box. It will look something like this for the x value:
    if (x<minX) {
   x = minX;
}
if (x>maxX) {
    x = maxX;
}

    See if you can figure out how to do this for the pupil's y value.
  12. Now we need to get the right eye to look the same.
  13. Find the difference (using subtraction) between the left eye's x-position and right eye's x-position, and remember the value.
  14. Also, find the difference (using subtraction) between the left eye's y-position and right eye's y-position, and remember the value.
  15. Draw a second black ellipse for the right pupil, making its position relative to the left pupil. Do this by setting the x and y values for the ellipse using (x + difference) and (y + difference). If the eyeballs are the same size, this should work.
  16. Make sure you SAVE YOUR CODE when you are done.
  17. Extra: Make the color of the eyes change when the user moves the mouse, by using the mouseX and mouseY variables in the fill() for the eyes.
