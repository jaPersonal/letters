Where's Waldo

Steps:

  1. Open wheres_waldo.pde using Processing.
  2. Choose a puzzle image from the "img" folder in wheres_waldo, then drag and drop it into your processing window
  3. On line 1 of your sketch (outside of the methods) declare a variable for the image: 
    PImage puzzle;
  4. In your setup() method, load your image using the following code: 
    puzzle = loadImage(“puzzle.jpg”);
  5. Change the code you just entered so that “puzzle.jpg” matches the name of the picture you chose.
  6. Set the size of your sketch and the size of your image to be the same by entering the following code in the setup method. 
    size(800,600);
puzzle.resize(width,height);
  7. You now need to imagine a rectangle around waldo. This is known as a 'hit box'. You will need to figure out the x and y numbers for two of the corners of the hit box.
    You can use the code below to print out where you click (like you did with the Terminator Cat's eyes). 
    if(mousePressed){
    println("Mouse’s x-position: " + mouseX + "\n" +
     "Mouse’s y-position: " + mouseY + "\n");
}

    First, click on the top left corner and save the values in variables:
    int minX;
    int minY;

    Next, click on the bottom right corner and save the values in variables:
    int maxX;
    int maxY;

  8. When the user clicks the mouse, we want to know if the user has found Waldo - that means they clicked inside of the hit box. To do this, we will need to check that the mouse's x value is between the minX and maxX of the box, and that the mouse's y value is also between the minY and maxY of the box. The code to do so will look something like this: 
    if(mousePressed){
   if(mouseX > minX && mouseX < maxX &&
         mouseY > minY && mouseY < maxY){
         
            text("You Found Waldo!", width/2, height/2);

}
    }
}
  9. EXTRA: Repeat the steps above to alert users when they identify additional characters within the puzzle. Depending on the puzzle you selected, additional characters are labeled with the following numbers:
    • Woof the dog...'s tail (#2)
    • Wilma, Waldo's girlfriend (#3)
    • Whitebeard the Wizard (#4)
    • Odlaw (or anti-Waldo) (#5)
